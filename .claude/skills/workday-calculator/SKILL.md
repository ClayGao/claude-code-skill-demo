# 工作日與工時計算器

## 功能概述

計算兩個日期之間的工作日數、加班時數，並自動排除週末和國定假日。

## 基本流程

1. 讀取用戶提供的日期範圍或工時記錄
2. 判斷計算類型（工作日/加班/假日）
3. 呼叫對應的計算腳本
4. 產出格式化的計算結果

## 支援的計算類型

| 類型   | 功能         | 說明               |
| ------ | ------------ | ------------------ |
| 工作日 | 計算工作日數 | 排除週末和國定假日 |
| 加班   | 計算加班時數 | 超過 8 小時的部分  |
| 假日   | 查詢假日清單 | 支援台灣國定假日   |

## 輸出格式

```
┌──────────────────┬────────────┐
│ 項目             │ 數值       │
├──────────────────┼────────────┤
│ 開始日期         │ 2024-01-01 │
│ 結束日期         │ 2024-01-31 │
│ 總天數           │ 31 天      │
│ 工作日           │ 22 天      │
│ 週末             │ 8 天       │
│ 國定假日         │ 1 天       │
└──────────────────┴────────────┘
```
